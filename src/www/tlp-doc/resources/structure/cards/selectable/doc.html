Most of the time you will allow the user to select a card among others. This is done by using the class .tlp-card-selectable.

It is recommended to put the .tlp-card on a <a> element, so that end user can use the whole card as a regular link (open in new tab, copy link location, â€¦) which is not possible with a <div> element.
