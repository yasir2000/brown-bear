• You can specify the dropdown menu element via dropdown_menu options. Otherwise TLP will look for a trigger's sibling with the tlp-dropdown-menu class.
  • You can right align the dropdown menu with the .tlp-dropdown-menu-right class.
  • You can top align the dropdown menu with the .tlp-dropdown-menu-top class.
  • Don't forget the .tlp-dropdown-menu-item-icon class if you use icons.
  • You can disable closing the dropdopwn on the escape key pressed: const the_dropdown = tlp.createDropdown(trigger_element, {keyboard: false} );
  • const the_dropdown = tlp.createDropdown(trigger_element); will return a Dropdown instance. You can then call show(), hide(), or toggle() methods on this instance.
  • On the the_dropdown.show() method an event tlp-dropdown-shown is dispatched.
  • On the the_dropdown.hide() method an event tlp-dropdown-hidden is dispatched.
