• When there are both cancel and primary action buttons, then the primary button is always on the right.
  • To be able to use large buttons on the footer you should use the class tlp-modal-footer-large.
  • Buttons that have data-dismiss="modal" attribute in the modal will close it.
  • Click on the backdrop or on the escape key will also close the modal.
  • Elements that have data-modal-focus attribute in the modal will take focus when it opens.
  • You must explicitly call the modal in javascript: var the_modal = tlp.createModal('element-id');. This will return a Modal instance. You can then call show(), hide(), or toggle() methods on this instance.
  • On the the_modal.show() method an event tlp-modal-shown is dispatched on both the modal and Document.
  • On the the_modal.hide() method an event tlp-modal-hidden is dispatched on both the modal and Document.

Arguments:

  • { Element id } id
  • (Optional) { Object } options. Contains:
    • { Boolean } keyboard, defaults to true

      When true, enables closing the modal when pressing the ESC key.

    • { Boolean } destroy_on_hide, defaults to false

      When true, destroys the event listeners when hiding the modal. Useful when for example you create new modals on a button click.
